# Automata — אתר תדמית

עמוד נחיתה סטטי של **Automata** — פתרונות AI ואוטומציה לעסקים.
בלי framework, בלי build — HTML/CSS/JS נקיים. כל שינוי שנדחף ל-main עולה לאוויר אוטומטית ב-Vercel.

## מבנה

| קובץ | מה יש בו |
|---|---|
| `index.html` | כל התוכן והטקסטים של העמוד |
| `css/style.css` | כל העיצוב (צבעים, ריווח, רספונסיב) |
| `js/main.js` | אנימציות, תפריט מובייל, קישורי CTA |
| `css/fonts.css` | פונט Heebo מוטמע — **לא לערוך** |

## עריכה מהירה

- **לשנות טקסט:** עורכים ישירות ב-`index.html` (הכול בעברית, מחולק לסקשנים עם הערות).
- **לשנות קישורי וואטסאפ / יומן:** בראש `js/main.js` — המשתנים `CAL` ו-`WA`.
- **לשנות צבעים:** בראש `css/style.css` — משתני ה-`:root`.

## פיתוח מקומי

```bash
npx serve .
# או פשוט לפתוח את index.html בדפדפן
```

## דיפלוי

Push ל-`main` ← Vercel בונה ומעלה אוטומטית. זהו.
