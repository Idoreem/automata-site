# Automata — אתר תדמית

עמוד נחיתה סטטי של **Automata** — פתרונות AI ואוטומציה לעסקים.
בלי framework, בלי build — HTML/CSS/JS נקיים. כל שינוי שנדחף ל-main עולה לאוויר אוטומטית ב-Vercel.

## המערכת העיצובית — "מרכז בקרה"

העמוד הוא לוח הבקרה של העסק: רקע קרם, טיפוגרפיה בדיו כהה, וכתום־סיגנל אחד.
החתימה היא **הקונסולה הכהה** ב-hero — ליד נכנס, נענה, מסונן, ומגיע ל-CRM מול העיניים.

| טוקן | ערך | לאן |
|---|---|---|
| `--paper` | `#FBF7EC` | הקרם — הרקע הראשי |
| `--ink` | `#151419` | כותרות |
| `--signal` | `#FF6B35` | הכתום. הדבר הנועז היחיד — אל תפזר אותו |
| `--console` | `#151118` | המשטח הכהה: הקונסולה, ההוכחה, ה-CTA הסופי, הפוטר |

הכל יושב ב-`:root` בראש `css/style.css`.

### שתי מלכודות שכדאי להכיר

1. **הגופן העברי (Trashim CLM) נטען במשקל אחד בלבד — 700.**
   `font-weight` על עברית לא יעשה שום דבר. ההיררכיה נבנית מגודל, צבע וריווח.

2. **`letter-spacing` ו-`text-transform:uppercase` הם כלים לטיניים.**
   על עברית הם שוברים את המילה (`א י ך  ז ה  ע ו ב ד`). הם שמורים למחרוזות
   לטיניות בלבד: `.status`, `.console-hd`, `.case .k`, `.fcol h4`, `.step .num`.

## מבנה

| קובץ | מה יש בו |
|---|---|
| `index.html` | כל התוכן והטקסטים, מחולק לסקשנים עם הערות |
| `css/style.css` | כל העיצוב. הטוקנים ב-`:root` בראש הקובץ |
| `js/main.js` | ניווט, תפריט מובייל, חשיפה בגלילה, ספירה עולה |
| `css/fonts.css` | Trashim CLM + Assistant, מוטמעים — **לא לערוך** (נוצר אוטומטית) |

## עריכה מהירה

- **טקסט:** ישירות ב-`index.html`.
- **קישורי וואטסאפ / יומן:** על תגיות ה-`<a>` ב-`index.html` (מסומנות `data-cal` ו-`data-wa`).
  הם **לא** מוזרקים מ-JS — כדי שכל כפתור יעבוד גם אם ה-JS נופל. אם משנים קישור,
  צריך לעדכן את כל המופעים (חיפוש `data-cal` / `data-wa`).
- **צבעים:** משתני ה-`:root` בראש `css/style.css`.
